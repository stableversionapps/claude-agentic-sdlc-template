# Market Research

Competitive intelligence and market analysis.

## Usage
`/market-research <topic>` — Research a specific topic
`/market-research compare <feature>` — Compare feature across competitors
`/market-research trends` — Latest industry trends

## Analysis Framework

### Competitor Tiers
- **Tier 1 (Direct):** Same market, same core problem
- **Tier 2 (Adjacent):** Overlapping features, different primary focus
- **Tier 3 (Ecosystem):** Platform-level players that could subsume features

### Analysis Dimensions
For each competitor:
1. Core value proposition
2. Pricing model
3. Platform coverage
4. UI/UX quality
5. Feature depth
6. AI/intelligence features
7. Privacy stance
8. Community size and sentiment
9. Key weaknesses (from user reviews)

### Differentiation Levers
1. Privacy advantage
2. Offline capability
3. Target market focus
4. AI without data harvesting
5. Technical superiority
6. Cross-platform coverage
7. Open/extensible architecture

## Output
- Feature comparison matrix
- Gap analysis with recommendations
- Market opportunity assessment
- Prioritized feature suggestions
